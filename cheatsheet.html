HTML

Cheatsheet HTML
<h1> The <h1> tag in HTML is used to define the most important heading on a webpage.
<p> The <p> tag in HTML is used to define a paragraph.
<div> The <div> tag in HTML is a block-level container used to group together HTML elements.
<span> The <span> tag in HTML is an inline container used to group together text or other inline elements.
Cheatsheet CSS
font-family The font-family property in CSS specifies the typeface to be used for the text within an element.
color The color property in CSS is used to set the color of the text within an element.
margin The margin property in CSS is used to create space around elements, outside of their borders.
position The position property in CSS specifies how an element is positioned in the document.
